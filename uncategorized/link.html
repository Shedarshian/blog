下面是一些库的document，只是因为我用过觉得还不错所以我存了网址而已，按语言分类 ### Latex宏包

Latex宏包我直接放ctan的链接了，里面点Package documentation直接就是详细文档。 + Latex图形环境介绍的中译本，不建议观看 Latex2e插图指南 + beamer user guide （PDF文件注意）著名的用latex做slide的宏包 + xparse 更加自定义化的指令 + etoolbox 主要面向写指令的，一些新增指令以及对背后系统如计数器等的操作 + ifthen 简单的条件分支和循环语句 + fancyvrb 更实用美观的verbatim（内嵌代码） + physics 更简洁实用地输入在物理里常用的符号，像狄拉克括号、向量分析、微分变分等等。 + chemfig 有机化学式 + circ 电路图 + musictex 五线谱（没人会用latex画五线谱吧。。。）

  • LatexTools Sublime text的latex plugin

#C++库

  • cpp reference C++官方文档，强烈建议都去读一读（我从头到尾读过一遍）
  • CLN 数值运算库，是下面的符号库的依赖
  • GiNaC 符号计算库（我还没用过），说实话我并不太推荐
  • ROOT 实验高能物理中使用的数据分析拟合与画图的包，内含一个C++解释器（但是不支持C++14，残念）
  • CLHEP 也是实验高能物理里比较实用的包

#python module

  • Python documentation python的官方文档，查内置库的时候很常用（还没完整的读过一遍），有中文但是这个中文现在的版本有点像机翻，没有C++的翻的好
  • six 用于py2与py3兼容的库
  • Wand 一个图片处理库
  • Pillow 另一个更好用的图片处理库
  • NumPy 数值运算库
  • SymPy 符号计算库，这两个也都比较有名
  • rootpy 更加pythonic的ROOT，还在更新中
